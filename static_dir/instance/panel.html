Welcome to dabr.ca, a microblogging platform powered by Pleroma.

This site supports multiple web frontends. Have a try:

  • Pleroma FE - the default one
  • MastoFE (glitch-lily) - based on Mastodon (experimental)
  • Soapbox - similar to Twitter
  • Pinafore - simple and clean UI
  • Sengi - similar to TweetDeck
